# E-Board
##### Eboard enables you to be aware of any up coming or currently happening events  and gigs  around you 
# Technologies used
- Firebase Authentication  
- Firebase RealtimeDatabase
- Google Places Api
- Google Location Api
- Google Geofencing Api
